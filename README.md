## ✨ Features

- 🔐 **JWT Authentication** — Secure user registration and login
- 🤖 **AI-Powered Reviews** — Google Gemini 2.5 Flash analyzes your code
- 📊 **Smart Scoring** — Get a score from 0-100 with detailed feedback
- 🔍 **4 Review Categories** — Security, Performance, Quality, and Bugs
- 📝 **Code Editor** — Monaco Editor with syntax highlighting
- 🌙 **Dark/Light Mode** — Toggle between themes
- 📤 **Export PDF** — Download reviews as PDF
- 📋 **Copy Code** — One-click copy for optimized snippets
- 🔎 **Search & Filter** — Filter history by language and score
- 📱 **Responsive** — Works on desktop, tablet, and mobile
